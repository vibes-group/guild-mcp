# guild-mcp

MCP-сервер для чтения и поиска сообщений Discord-сервера.

## Подключение

### Claude Code

```
claude mcp add --transport http guild-mcp <url>
```

### Codex

```
codex mcp add guild-mcp --url <url>
```
